Project Description:

This project is a simplified real-time flight-control subsystem that could be built by Lockheed Martin, built on the ESP32 using FreeRTOS. The system reads sensor data from a simulated IMU (potentiometer), computes actuator commands (simulating jet engine activity), and responds instantly to an emergency-override button using a hardware interrupt. A Wi-Fi control panel allows manual triggering and clearing of emergency states, perhaps by a ground team if the pilot is incapacitated. The goal of this prototype is to demonstrate how deterministic timing, task scheduling, synchronization primitives, and interrupts work together to meet real-time deadlines in a safety-critical context.
